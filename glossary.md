# 用語集（Glossary）

英語の固有名詞と日本語での意味をまとめた参照ページです。
本サイトでは固有名詞は基本的に**英語表記**で統一しています。

---

## クラス（Classes）

| 英語 | 意味・説明 |
|------|-----------|
| **Oracle** | 魔法系クラス。電気属性スキルを中心に持つ |
| **Barbarian** | 近接戦士クラス。物理・氷属性スキルを持つ |
| **Knight** | タンク系クラス。火属性スキルを持つ |
| **Tracker** | 遠距離クラス。物理・毒属性スキルを持つ |

---

## Wyrmling（ワイバーミング）

相棒となるドラゴン。4種類あり、クラスと自由に組み合わせられる。

| 英語 | 属性 |
|------|------|
| **Crimson** | 火 |
| **Pale** | 氷 |
| **Viridian** | 毒 |
| **Cobalt** | 電気 |

---

## ゲームシステム

| 英語 | 意味・説明 |
|------|-----------|
| **Ancestral Grid** | 六角形グリッド上にスキルを配置してビルドを組むシステム。Lv.50で全スロット解放 |
| **Ancestral Dragon** | エンドゲームの最終目標となる4体のドラゴン |
| **Fragment** | Ancestral Gridに配置するスキルピース |
| **Talent** | Ancestral Gridのパッシブボーナス |
| **Innate** | 各クラス固有の特性・パッシブ能力 |
| **Adaptive Damage** | 最も高い属性ボーナスが全スキルに自動適用される仕組み |
| **Loot Filter** | 不要アイテムを自動変換するフィルター設定 |

## 通貨・素材

| 英語 | 意味・説明 |
|------|-----------|
| **Gold** | 基本通貨。装備の売買・クラフトに使用 |
| **Dust** | 上位通貨。神話レアリティのクラフト・強化に使用 |
| **Material** | クラフト素材（Common / Divine の2種） |
| **Resource** | クラフト素材（ポーション・リング用） |
| **Dark Merchant** | 装備を売却・Dustに変換できるNPC |
| **Grand Benedicteur** | ルートフィルターで自動変換できるアイテム |
